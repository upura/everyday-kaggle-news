# グラフ

グラフ構造データ（ネットワーク、分子、関係データなど）を扱う話題のページです。
表やテキストとは異なるモデリング（グラフニューラルネットワークなど）が必要になります。

## 押さえどころ

- グラフデータは「ノードとエッジ」という構造そのものが情報を持つため、GBDT に落とすための特徴量化（集約や埋め込み）か、GNN による直接のモデリングかが最初の分岐になる
- 体系的な学習にはグラフアルゴリズムの演習教材が役立つ
