# 入門者・初学者向け

Kaggle を始めたばかりの人向けの資料をまとめたページです。
Kaggle の全体像は「[入門者・初学者向けガイド](../../quickstart.md)」を先に読むのがおすすめです。

## 押さえどころ

- 最初の一歩は Titanic などのチュートリアルコンペだが、その先で何をやるかが続けられるかの分かれ目。定番記事や入門動画で「次にやること」の道筋をつかむ
- 公式の [Kaggle Learn](https://www.kaggle.com/learn) で基礎を短時間で確認できる
- 書籍でまとめて学ぶ選択肢は[書籍一覧](../../books.md)を参照
- 学び続けるための姿勢は[心構え](./mindset.md)も参考になる
