# 数理最適化・ヒューリスティック

Kaggle の Santa コンペに代表される、機械学習ではなく組合せ最適化・ヒューリスティックで競う形式です。
AtCoder のヒューリスティックコンテスト(AHC)と技術的な親和性が高い領域です。

## 押さえどころ

- Santa 系コンペは巡回セールスマン問題などの組合せ最適化が題材で、焼きなまし法・ビームサーチなどのヒューリスティックと、ソルバー(混合整数計画など)の使い分けが軸になる
- AtCoder(特に AHC)での経験・練習がそのまま役立つ。入門には AtCoder Heuristic First-step の講義スライドが充実している
- 機械学習コンペとは評価・提出の性質が異なり、探索の実装力と計算資源の使い方が効く
