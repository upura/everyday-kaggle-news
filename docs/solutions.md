# コンペ解法

本ページでは、コンペの解法に関する情報を掲載します。

## Kaggle 公式

- Discussion: Kaggle コンペごとに、参加者が投稿した解法が Discussion で閲覧できます。順位表の「Solution」から個別の投稿に遷移できます。
- [YouTube](https://www.youtube.com/kaggle): 定期的に解法動画が公開されています。
